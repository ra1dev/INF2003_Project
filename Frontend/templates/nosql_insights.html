MongoDB Role

This page demonstrates aggregation over document data using nested event arrays and denormalized player-match documents.

PostgreSQL remains the source for normalized multi-season match, team, and player-season summaries.

Query Techniques

Uses MongoDB aggregation stages such as $unwind, $group, $sort, and $facet.
